Caption to Image Generation

Overview

Artificial image generation is becoming more and more popular and, as of recently, has become easy to use, widely available, and giving high quality results. We each were able to download and run code from a research paper of our choice that used different methods to get the same result. We then compared our results, giving our models the same prompt to generate and give the pros and cons of each model

Models

Stable Diffusion

Implemented by Brandon Muller

Stable Diffusion is a latent text-to-image diffusion model capable of generating photo-realistic images given any text input, cultivates autonomous freedom to produce incredible imagery, empowers billions of people to create stunning art within seconds.

Stable Diffusion Results

Dream Booth

Implemented by Tiba Al Anssari

DreamBooth is a stable diffusion model that works by taking in a subject input images and then generating images of the subject in different prompts. It uses a process called fine-tuning to generate those images.

Dream Booth Results

Latent Diffusion

Implemented by Abie Faz

My model used the LAION-400M dataset and filtered out the NSFW text-image pairs. It also took it two repositories, one that used latent diffusion to gather text-image pairs via nearest neighbor lookup. The second repository took in the outputs of the first repo and made the image even clearer.
